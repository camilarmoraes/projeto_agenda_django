# projeto_agenda_django
Repositório contendo pequeno projeto de aprendizagem do Django Framework.
Para isto, foi utilizado procedimentos básicos do django:
  - Models
  - Organização da seção Admin
  - Levantamento de erros
  - Paginação, ordenação e filtragem de valores
  - Django Messages
  - CRUD completo
  - Login
  - Formulário nos models
